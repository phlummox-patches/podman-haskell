# podman

[![Hackage](https://img.shields.io/hackage/v/podman.svg)](https://hackage.haskell.org/package/podman)

A client library for [podman](https://podman.io)

## Features

- JSON decoder for API endpoints.
- HTTP client helper functions.

## Contribute

Run tests:

```
$ cabal build
$ cabal test
```
